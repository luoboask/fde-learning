# FDE 知识地图 — Gemini 生成 Prompt 集合

```
  通信：AllToAll（路由 token）| 扩展性最好

  CP 上下文并行：
  超长序列分段处理，Ring Attention
  通信：Ring AllReduce | 适合 128K+ 长上下文

- 右侧决策框：
  70B 单请求 → TP=8（8卡 NVLink）
  多请求高吞吐 → DP + TP 混合
  MoE 模型 → TP + EP
  超长上下文 → CP

- 底部："70B 模型最常用 TP=8，8 卡 A100/H100 NVLink 域内"
- 左上角标签 "FDE 分布式推理"
```
