# 对比与附录

> 本页面汇总 Claude Code 与其他同类工具的对比分析

## 1. 记忆系统对比

| 特性 | Claude Code | GitHub Copilot | Cursor |
|------|-------------|----------------|--------|
| 跨会话记忆 | `.md` 文件 | 无 | 项目索引 |
| 自动提取 | 后台分析 | 无 | 部分 |
| 记忆老化 | 时间衰减 | 不适用 | 无 |
| 透明性 | 完全可见 | 不透明 | 部分 |

## 2. 工具系统对比

| 特性 | Claude Code | Copilot CLI | Cursor |
|------|-------------|-------------|--------|
| 统一接口 | 是 (`Tool.ts`) | 否 | 否 |
| 权限系统 | 三级 | 二级 | 无 |
| MCP 支持 | 是 | 部分 | 否 |
| 安全沙箱 | 24 层检查 | 基础 | 无 |

## 3. 架构对比

| 特性 | Claude Code | Copilot CLI | Cursor |
|------|-------------|-------------|--------|
| 渲染引擎 | Ink (React) | 纯文本 | Electron |
| 状态管理 | 发布订阅 | 无 | Redux |
| 扩展系统 | Skills + Plugins | Extensions | Extensions |
| 代码量 | 51 万行 | 未公开 | 未公开 |
