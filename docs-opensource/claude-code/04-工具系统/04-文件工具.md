# 文件工具

> 源码: `tools/FileReadTool/` | `tools/FileWriteTool/` | `tools/FileEditTool/`

## 1. 文件读取

`FileReadTool`：读取文件内容，支持行号范围指定。

## 2. 文件写入

`FileWriteTool`：创建或覆盖文件内容。需要权限确认。

## 3. 文件编辑

`FileEditTool`：精确替换文件中的文本段。使用 `old_string` / `new_string` 模式，支持上下文定位。

## 4. Notebook 编辑

`NotebookEditTool`：编辑 Jupyter Notebook 的单元格。
