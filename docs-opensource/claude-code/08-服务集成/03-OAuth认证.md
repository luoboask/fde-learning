# OAuth 认证

> 源码: `services/oauth/`

## 1. 认证流程

```
用户运行 claude --login
    │
    ↓
┌──────────────────────┐
│ 启动本地回调服务器    │
│ (localhost:xxx)      │
└──────────┬───────────┘
           │
           ↓
┌──────────────────────┐
│ 打开浏览器授权页面    │
│ (Anthropic OAuth)    │
└──────────┬───────────┘
           │
           ↓
┌──────────────────────┐
│ 用户授权              │
│ → 重定向到本地回调    │
└──────────┬───────────┘
           │
           ↓
┌──────────────────────┐
│ 接收 code             │
│ → 交换 access_token   │
│ → 保存到本地存储      │
└──────────────────────┘
```

## 2. Token 管理

- Token 刷新（自动）
- 过期检测
- 401 错误处理（自动重新授权）
