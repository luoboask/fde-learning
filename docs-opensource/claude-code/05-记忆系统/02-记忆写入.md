# 记忆写入

> 源码: `memdir/memdir.ts` | `services/extractMemories/`

## 1. 写入机制

记忆通过 Claude 自身的输出写入文件系统。模型在对话中生成 `.md` 文件内容，通过 FileWriteTool 保存到记忆目录。

## 2. 写入规范

每个记忆文件遵循 frontmatter 格式：

```markdown
---
name: {{memory name}}
description: {{one-line description}}
type: {{user, feedback, project, reference}}
---

{{memory content}}
```

## 3. 后台自动提取

`services/extractMemories/` 定期分析对话历史，自动提取有价值的信息保存为记忆。
