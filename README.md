# FDE 学习中心

AI 前沿部署工程师（Frontier Deployment Engineer）—— 从入门到面试的系统学习路径。

## 在线文档

访问 [FDE 学习中心](https://luoboask.github.io/fde-learning/) 在线阅读。

## 本地运行

```bash
npm install
npm start
```

打开 http://localhost:3000 即可查看。
