实验：量化部署工作流

目标​

把 7B 模型从 FP16 量化到 INT8，验证精度和性能差异。

环境​

  • GPU：A100-80G 或 RTX 4090
  • Python >= 3.10

步骤​

1. 安装量化工具​

pip install autoawq vllm

2. 使用预量化模型​

# 直接从 HuggingFace 拉取 INT4 量化版本
vllm serve hugging-quants/Meta-Llama-3-8B-Instruct-AWQ-INT4 \
  --quantization awq \
  --host 0.0.0.0 \
  --port 8000

3. 对比 FP16 vs 量化版​

指标FP16INT4 (AWQ)
模型权重显存16 GB4 GB
推理速度基准略慢（需反量化）
MMLU 分数基准下降 1-3%
最大 batch受限可大幅增大
