Step 2пјҡеҗһеҗҗйҮҸе»әжЁЎвҖӢ

еҚ• GPU еҗһеҗҗи®Ўз®—вҖӢ

еҚ• GPU жҖ»еҗһеҗҗ = зҗҶи®әеі°еҖјеҗһеҗҗ Г— еҲ©з”ЁзҺҮзі»ж•° Г— batch_size

е…¶дёӯпјҡ
  зҗҶи®әеі°еҖјеҗһеҗҗпјҡH100 FP16 decode ~1000 tokens/sпјҲеҚ•иҜ·жұӮпјү
  еҲ©з”ЁзҺҮзі»ж•°пјҡContinuous Batching дёӢ ~0.7-0.85
  batch_sizeпјҡе№¶еҸ‘иҜ·жұӮж•°пјҲ4-32пјҢеҸ–еҶідәҺжЁЎеһӢеӨ§е°Ҹпјү

дёҚеҗҢжЁЎеһӢи§„жЁЎзҡ„еҗһеҗҗдј°з®—пјҲH100 еҚ•еҚЎпјүпјҡ

жіЁж„Ҹпјҡdecode йҳ¶ж®өпјҲйҖҗ token з”ҹжҲҗпјүжҳҜ compute-boundпјҢprefill йҳ¶ж®өжҳҜ memory-boundгҖӮе®һйҷ…з”ҹдә§дёӯд»Ҙ decode еҗһеҗҗдёәдё»гҖӮ

жҖ»еҗһеҗҗи®Ўз®—вҖӢ

йӣҶзҫӨжҖ»еҗһеҗҗ = еҚ•еҚЎеҗһеҗҗ Г— GPU ж•°йҮҸ Г— йӣҶзҫӨж•ҲзҺҮ

йӣҶзҫӨж•ҲзҺҮпјҡ
  еҚ•еҚЎйғЁзҪІпјҡ100%
  еӨҡеҚЎ TPпјҡ90-95%пјҲNCCL йҖҡдҝЎејҖй”Җ 5-10%пјү
  еӨҡиҠӮзӮ№ TPпјҡ80-90%пјҲи·ЁиҠӮзӮ№йҖҡдҝЎејҖй”Җ 10-20%пјү

и®Ўз®— GPU ж•°йҮҸвҖӢ

жүҖйңҖ GPU = ceil(еі°еҖј QPS Г— е№іеқҮиҫ“еҮә tokens Г— decode е»¶иҝҹ / еҚ• GPU жҜҸз§’ token еҗһеҗҗ)

Step 3пјҡйў„з•ҷ BufferвҖӢ

Buffer зұ»еһӢвҖӢ

Buffer зұ»еһӢжҜ”дҫӢз”ЁйҖ”
жӯЈеёё Buffer+30%еә”еҜ№ж—ҘеёёжіўеҠЁгҖҒеҚ•еҚЎж•…йҡң
еі°еҖј Buffer+50-100%еә”еҜ№зӘҒеҸ‘жөҒйҮҸпјҲйңҖеј№жҖ§ж–№жЎҲпјү
е®№зҒҫ BufferN+1д»»ж„ҸдёҖеҸ°жңәеҷЁж•…йҡңдёҚеҪұе“ҚжңҚеҠЎ

е®һдҫӢз»„еҗҲзӯ–з•ҘвҖӢ

жҖ» GPU = еҹәзЎҖиҙҹиҪҪ (60%) + еј№жҖ§жҢүйңҖ (25%) + еј№жҖ§ Spot (15%)

зӨәдҫӢпјҡйңҖиҰҒ 100 GPU
  - еҹәзЎҖпјҡ60 еј йў„з•ҷе®һдҫӢпјҲжңҲиҙ№жү“ 4 жҠҳпјү
  - еј№жҖ§пјҡ25 еј жҢүйңҖе®һдҫӢпјҲйҡҸж—¶еҗҜеҒңпјү
  - Spotпјҡ15 еј  Spot е®һдҫӢпјҲжҲҗжң¬жңҖдҪҺзҡ„еј№жҖ§пјү

еі°еҖјжөҒйҮҸеӨ„зҗҶвҖӢ

зӘҒеҸ‘жөҒйҮҸ vs е№іеқҮжөҒйҮҸвҖӢ

еә”еҜ№зӯ–з•Ҙзҹ©йҳөвҖӢ

и¶…еҮәжҜ”дҫӢзӯ–з•Ҙз”ЁжҲ·дҪ“йӘҢеҪұе“Қ
< 20%жҺ’йҳҹ + жөҒејҸиҫ“еҮәе»¶иҝҹд»Һ 500ms вҶ’ 1sпјҢеҸҜжҺҘеҸ—
20-50%еҝ«йҖҹеј№жҖ§жү©е®№пјҲжҢүйңҖе®һдҫӢпјүеҶ·еҗҜеҠЁ 30-120sпјҢжңҹй—ҙе»¶иҝҹеҚҮй«ҳ
> 50%йҷҗжөҒ + йҷҚзә§йғЁеҲҶиҜ·жұӮиў«жӢ’з»қжҲ–иҝ”еӣһз®ҖеҢ–з»“жһң

дёҚеҗҢ SLA иҰҒжұӮдёӢзҡ„е®№йҮҸи§„еҲ’вҖӢ

  • Batch Sizeпјҡ16-32пјҲжңҖеӨ§еҢ–еҗһеҗҗпјү
  • жҺ’йҳҹжңәеҲ¶пјҡе…Ғи®ёжҺ’йҳҹпјҢз”ЁжөҒејҸиҫ“еҮәж”№е–„ж„ҹзҹҘе»¶иҝҹ

йқўиҜ•и§Ҷи§’вҖӢ

йқўиҜ•е®ҳеҸҜиғҪй—®пјҡ

  1. "еі°еҖј 1000 QPSпјҢжҜҸдёӘиҜ·жұӮе№іеқҮ 100 output tokensпјҢ70B жЁЎеһӢйңҖиҰҒеӨҡе°‘ H100пјҹ"

    • еҚ• H100 decode еҗһеҗҗпјҲ70B, batch=8пјүпјҡ~350 tokens/s
    • жҖ»йңҖжұӮпјҡ1000 Г— 100 = 100,000 tokens/s
    • йңҖиҰҒ GPUпјҡ100,000 / 350 = ~286 еҚЎ
    • TP=2 жҜҸжЁЎеһӢе®һдҫӢпјҡ286 / 2 = 143 дёӘжЁЎеһӢе®һдҫӢ
    • иҖғиҷ‘еҲ©з”ЁзҺҮпјҲ70%пјү+ N+1пјҡ143 / 0.7 + 1 вүҲ 205 дёӘе®һдҫӢ вҶ’ 410 еј  H100

  2. "P99 е»¶иҝҹи¶…ж ҮжҖҺд№ҲжҺ’жҹҘпјҹ"

    • е…ҲзңӢ TTFTпјҲTime to First Tokenпјүпјҡprefill жҳҜеҗҰиҝҮж…ў
    • еҶҚзңӢ TPOTпјҲTime Per Output Tokenпјүпјҡdecode жҳҜеҗҰеҸ— KV Cache жҲ– batch еҪұе“Қ
    • жЈҖжҹҘжҳҜеҗҰжңүиҜ·жұӮжҺ’йҳҹпјҲqueue timeпјүпјҡGPU жҳҜеҗҰиҝҮиҪҪ
    • еҢәеҲҶжҳҜжЁЎеһӢжҺЁзҗҶж…ўиҝҳжҳҜзҪ‘з»ң/еәҸеҲ—еҢ–ж…ў

  3. "е®№йҮҸи§„еҲ’зҡ„ж ёеҝғе…¬ејҸжҳҜд»Җд№Ҳпјҹ"

    • GPU ж•°йҮҸ = (еі°еҖј QPS Г— е№іеқҮиҫ“еҮә tokens) / (еҚ• GPU жҜҸз§’ token еҗһеҗҗ Г— еҲ©з”ЁзҺҮ)
    • е…ій”®жҳҜеҮҶзЎ®жөӢйҮҸ"еҚ• GPU еҗһеҗҗ"пјҲз”Ёе®һйҷ…еҺӢжөӢпјҢдёҚз”ЁзҗҶи®әеі°еҖјпјү

жңҖдҪіе®һи·өвҖӢ

  1. е…ҲеҺӢжөӢеҶҚи§„еҲ’пјҡдёҚиҰҒзӣёдҝЎзҗҶи®әеҖјпјҢз”Ё vLLM benchmark е®һжөӢ
  2. жҢүеі°еҖјзҡ„ 1.5x и§„еҲ’пјҡз•ҷ 50% дҪҷйҮҸеә”еҜ№еўһй•ҝе’ҢејӮеёё
  3. еҲҶйҳ¶ж®өйғЁзҪІпјҡе…ҲдёҠ 60% зҡ„е®№йҮҸпјҢж №жҚ®е®һйҷ…ж•°жҚ®еҶҚжү©е®№
  4. е®ҡжңҹеӨҚиҜ„пјҡжҜҸеӯЈеәҰйҮҚж–°иҜ„дј° QPS и¶ӢеҠҝе’Ң GPU еҲ©з”ЁзҺҮ
  5. е…іжіЁй•ҝе°ҫпјҡP99 е»¶иҝҹжҜ”е№іеқҮе»¶иҝҹжӣҙиғҪеҸҚжҳ з”ЁжҲ·дҪ“йӘҢ
  6. еј№жҖ§дјҳе…ҲпјҡдёҺе…¶дёҖж¬ЎжҖ§д№°еӨҹжүҖжңү GPUпјҢдёҚеҰӮйў„з•ҷ + Spot з»„еҗҲ
