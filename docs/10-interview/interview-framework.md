# 面试答题框架

## 面试结构

| 轮次 | 考察重点 |
|------|----------|
| 一面 | 技术深度：推理引擎、GPU |
| 二面 | 项目经验：线上部署、踩坑 |
| 三面 | 技术规划、前沿跟进 |
| 交叉面 | 广度：分布式、网络 |
| HR面 | 动机、稳定性 |

## 通用答题框架

```
技术题：分析瓶颈 → 列出方案 → 给出数据
架构题：澄清需求 → 画架构 → 讨论挑战 → 演进路线
成本题：拆解成本 → 优化手段 → 量化结果 → 复盘
行为题：Situation → Task → Action → Result
```
