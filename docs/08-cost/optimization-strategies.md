# 成本优化策略

## 分层优化

| 层级 | 策略 | 效果 |
|------|------|------|
| 模型层 | 量化、蒸馏 | 省 30-70% GPU |
| 引擎层 | Continuous Batching | 省 50-70% GPU |
| 基础设施 | Spot 实例 + 预留 | 省 50-75% |

## 自建 vs 云 API

```
月调用量 < 10 万   → 云 API
月调用量 10 万-1000 万 → 混合
月调用量 > 1000 万  → 自建
```
