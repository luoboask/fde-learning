# 分布式推理概述

## 为什么需要

- 单 GPU 放不下大模型（70B = 140GB）
- 单 GPU 性能不够（QPS 要求高）

## 四种并行策略

| 策略 | 切分维度 | 通信要求 | 适用 |
|------|----------|----------|------|
| DP | 请求 | 无 | 小模型 |
| TP | 层内矩阵 | NVLink | 大模型 |
| PP | 层间 | PCIe 可行 | 超大模型 |
| Expert | Expert | 网络 | MoE |
