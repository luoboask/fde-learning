# 知识管理

## 知识库结构

```
知识库/
├── 模型卡片/        # 每个模型的部署信息和性能数据
├── 性能基线/        # 各 GPU 型号上的性能基准
├── 最佳实践/        # 部署 checklist、优化手册、SOP
├── 踩坑记录/        # OOM 排查、高延迟案例
├── 技术评估报告/    # 新技术评估
└── 培训资料/        # onboarding 文档、技术分享
```

## 模型卡片

每个模型一张卡片：参数量、架构、GPU 配置、性能基线、已知问题、优化建议。

## 知识传承

- 新人 onboarding 文档持续更新
- 离职交接：知识沉淀到文档
- 每季度审计知识库
