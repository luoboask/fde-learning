# FDE 人才画像

## 五维能力模型

```
           AI 理解力
            /\
           /  \
  工程能力 ------- 运营意识
        \         /
         \       /
        学习力 × 协作力
```

## 理想候选人

- 2-5 年后端开发经验（Java/Go/Python）
- 对 AI 有热情（自己跑过模型）
- 有 Linux/系统编程基础
- 有持续学习的习惯

## 不适合的人

- 纯算法研究员（不关心工程落地）
- 纯业务开发（不懂底层原理）
- 纯运维（不会开发）
- 工具收集癖（没有深度）
