# 培养机制

## 技术轮转制（每月）

每月选定一个技术主题，每人 deep dive 一个子方向：

| 月份 | 主题 |
|------|------|
| 1月 | 推理引擎源码 |
| 2月 | 量化技术 |
| 3月 | 批处理优化 |
| 4月 | 分布式推理 |
| 5月 | GPU 底层 |
| 6月 | 可观测性 |
| 7月 | 前沿技术 |
| 8月 | 新模型适配 |
| ... | ... |

## 实战训练营（每季度）

真实场景挑战：新模型上线、性能优化、故障演练、架构设计。

## 1v1 机制（每两周）

技术成长、职业目标、当前挑战、个人状态。

## 前沿跟踪

周度轮值整理、月度深度评估、季度技术雷达更新。
