# 三级成长路径

## L1：入门 FDE（0-6 个月）

**目标：** 能独立完成单模型部署和基础优化

- 学习 Transformer、vLLM、GPU 基础、量化
- 实践：部署 7B/13B 模型，做性能分析
- 考核：能独立部署、看懂 profiling、写好文档

## L2：成熟 FDE（6-18 个月）

**目标：** 能负责生产级推理服务，独立完成优化

- 深入推理引擎源码、分布式推理、高级量化
- 实践：独立优化项目、新模型适配、容量规划
- 考核：有量化优化成果、能处理线上故障

## L3：高级/Lead FDE（18 个月+）

**目标：** 能设计推理平台架构，带领团队

- 自定义 kernel、推理引擎二次开发
- 公司级推理平台设计、跨部门推动技术方案
