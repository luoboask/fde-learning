# 模型量化基础

## 为什么需要量化

```
70B 模型：
  FP16：140 GB → 需要 2 张 A100-80G
  INT8：70 GB  → 1 张 A100-80G 就够了
  INT4：35 GB  → 1 张 A100 还能跑更大 batch
```

**量化 = 用低精度表示高精度数值，核心是降低显存占用和提升推理速度。**

## 主流量化方案对比

| 方案 | 精度 | 速度 | 压缩率 | 硬件要求 |
|------|------|------|--------|----------|
| AWQ | ★★★★★ | ★★★ | ★★★★ | 通用 |
| GPTQ | ★★★★ | ★★★ | ★★★★★ | 通用 |
| SmoothQuant | ★★★★ | ★★★★ | ★★★ | 通用 |
| FP8 | ★★★★ | ★★★★★ | ★★★ | H100+ |

## 量化决策流程

1. **目标**：省显存 → INT4/INT8；提速度 → FP8/SmoothQuant
2. **硬件**：H100 → FP8；A100 → INT8
3. **验证**：跑 benchmark，确认精度损失可接受
